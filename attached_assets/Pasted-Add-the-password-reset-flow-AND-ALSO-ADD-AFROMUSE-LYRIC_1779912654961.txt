🚀 AFROMUSE LYRICS INTELLIGENCE V6

(HOOK ENGINE + ADLIB INTELLIGENCE SYSTEM)
🎯 1. HOOK ENGINE (MEMORABILITY LAYER)
PURPOSE

Force every generated song to contain a strong, repeatable, viral-ready hook.

This is not optional.
Every song MUST have a primary hook line.

🧠 CORE RULE

A song is considered valid ONLY if:

it contains a clear central hook phrase
that hook is repeated in the chorus
and is easy to remember after one listen
🧩 HOOK STRUCTURE RULES
1. PRIMARY HOOK LINE
Must be short (4–8 words)
Must be emotionally direct
Must be phonetically catchy

Example structure:

“Lagos rain wash my hope away”
“No money, but I still dey smile”
“Tonight we no dey sleep”
2. REPETITION RULE
Hook must appear:
at least 2–4 times in chorus
Slight variations allowed ONLY in later chorus
3. PHONETIC CATCHINESS

Hook must include at least ONE of:

repetition (“fall down, fall down”)
bounce words (“oh no”, “yeah yeah”, “mmh mmh”)
vowel-heavy phrasing (easy to sing)
4. HOOK DOMINANCE
Chorus must revolve around hook
No chorus line should overshadow hook
5. VIRAL TEST (MANDATORY)

Hook must pass:

easy to chant
easy to remember
understandable in one listen
❌ FORBIDDEN
long complex sentences as hook
multiple competing hooks
weak emotional statements
storytelling inside hook (hook is NOT a verse)
🧠 OUTPUT RULE

System must:

extract or generate a keeper line
reinforce it across chorus
evolve it slightly across repeated choruses
🎤 2. ADLIB INTELLIGENCE SYSTEM
(WAVE + RESPONSE CONTROL)
PURPOSE

Simulate real Afrobeat vocal performance using:

ad-libs
echoes
crowd-response energy
🧠 CORE RULE

Every section must feel like a performed record, not written text.

🧩 ADLIB TYPES
1. ECHO ADLIB (most common)

Repeats last phrase emotionally

Example:

“I dey fall (fall down)”
“No money (no money)”
2. EMOTIONAL ADLIB

Adds feeling layer

Examples:

(oh no)
(God abeg)
(chai)
(yeah yeah)
3. RESPONSE ADLIB

Creates call-and-response effect

Example:

“Who go save me? (nobody)”
“We go make am? (we go!)”
4. FILLER ADLIB

Adds rhythm and groove

Examples:

(uhh)
(mmh)
(ayy)
(yo)
🎚️ PLACEMENT RULES
VERSES
Light usage
1 ad-lib every 2–4 lines
CHORUS
Heavy usage
1–2 ad-libs per line allowed
must enhance rhythm, not clutter
BRIDGE
Emotional ad-libs ONLY
deeper, slower, more expressive
⚠️ BALANCE RULE

Ad-libs must:

support main lyric
never overpower it
never appear on EVERY line
🎵 FLOW INTEGRATION RULE

Ad-libs must:

land on pauses
match rhythm spacing
feel like natural breath or echo
❌ FORBIDDEN
random ad-libs with no meaning
repeating same ad-lib too often
placing ad-libs mid-sentence awkwardly
overcrowding lines
🧠 OUTPUT RULE

System must:

dynamically insert ad-libs based on section type
vary ad-lib types across song
align ad-libs with vocal rhythm (Flow Engine)
