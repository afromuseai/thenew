You are a senior full-stack engineer building a production-ready AI music generation system.

Your task is to integrate an external AI Music Generation API into an existing app. The system must support generating instrumental beats and full songs, handling async jobs, and returning playable audio to the frontend.

DO NOT use placeholder logic. Build real, working architecture with clean separation of concerns.

---

## 🧠 SYSTEM OVERVIEW

We are building a feature called:

👉 “Generate Instrumental / Song”

User inputs:

* genre
* mood
* bpm (optional)
* key (optional)
* energy
* sound reference (optional)
* production style (optional)
* lyrics (optional, structured into sections)

System must:

1. Send request to AI Music API
2. Receive a task_id
3. Track job status (polling OR callback)
4. Return final audio URL
5. Display playable audio in UI

---

## ⚙️ BACKEND REQUIREMENTS (Node.js / TypeScript)

### 1. Create Service Layer

Create a service:

`services/musicGeneration.ts`

Responsibilities:

* Build request body from user input
* Handle two modes:

  * Instrumental (no lyrics)
  * Full song (with lyrics)
* Send POST request to start generation
* Extract task_id
* Poll for completion
* Return final result

---

### 2. Polling System

Implement a robust polling loop:

* Retry every 5–6 seconds
* Max attempts: 40
* Stop when:

  * status = completed
  * OR audio_url exists
* Handle failure states

Return:

```ts
{
  audioUrl: string
  title?: string
  coverArt?: string
}
```

---

### 3. Optional Callback System (Advanced)

Add support for webhook callback:

* Endpoint: `/api/music/callback`
* Store results in memory (Map or store)
* If callback result exists → stop polling early

---

### 4. Controller Layer

Create route:

`POST /api/music/generate`

Flow:

1. Validate input
2. Call service
3. Return response:

```json
{
  success: true,
  data: {
    audioUrl,
    title,
    coverArt
  }
}
```

---

### 5. Error Handling

* Handle API failures gracefully
* Timeout handling
* Return clean errors:

```json
{
  success: false,
  error: "Generation failed"
}
```
